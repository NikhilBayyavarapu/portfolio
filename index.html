Server DataStore Support for Widgets in MATLAB

This project leverages the capabilities of MATLAB widgets to visualize and display data on the client side. However, certain resource-intensive operations cannot be efficiently performed on the client. To overcome this limitation, the project introduces a Server DataStore. The heavy computational tasks will be performed on the server. The processed information will then be transmitted back to the client for rendering and presentation using the widget. This approach ensures that the client's resources are not overloaded and allows for seamless data visualization.

Technologies: Vanilla JS, MATLAB
